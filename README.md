## Featured Repositories

**[Microsoft_Sentinel](https://github.com/madhuperera/Microsoft_Sentinel)**
Detection rules, KQL investigation queries, dashboards, and watchlists for Microsoft Sentinel. Organized by security threat domain. Actively maintained.

**[CA-Automation-01](https://github.com/madhuperera/CA-Automation-01)**
Infrastructure-as-Code automation for Microsoft Entra ID Conditional Access policies. Provides a structured baseline using PowerShell and ARM templates.

**[naming-conventions](https://github.com/madhuperera/naming-conventions)**
Practical naming convention guidance for Azure, Microsoft 365, Intune, Entra ID, and Defender XDR resources. Based on enterprise implementations.
